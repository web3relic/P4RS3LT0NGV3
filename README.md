# P4RS3LT0NGV3 Dashboard

This project now features a modern dashboard UI cloned from [square-ui dashboard-1 template](https://github.com/ln-dev7/square-ui/tree/master/templates/dashboard-1).

## Project Structure

- **Root directory**: Contains the new Next.js dashboard application
- **/backup**: Contains all the previous P4RS3LT0NGV3 code (will be integrated later)

## Tech Stack

- **Next.js 16** - React framework for production
- **React 19** - UI library
- **Tailwind CSS 4** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript
- **Radix UI** - Accessible component primitives
- **Recharts** - Charting library
- **Zustand** - State management
- **Lucide React** - Icon library

## Getting Started

### Prerequisites

This project uses `pnpm` as the package manager. If you don't have it installed:

```bash
npm install -g pnpm
```

### Installation

1. Install dependencies:
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Other Commands

- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint
